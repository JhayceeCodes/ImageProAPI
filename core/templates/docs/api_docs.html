ImageProAPI Documentation

ImageProAPI is a REST API that allows clients to upload images and apply processing operations such as resizing, compression, filters, and format conversion. Images are processed asynchronously and can be downloaded once processing is complete.

Authentication

This API uses JWT (JSON Web Tokens) for authentication. Authenticated users can store and retrieve their processed images. Anonymous users can upload images but with stricter limits.

Endpoints

  • POST /accounts/register/ — Create a new user account
  • POST /accounts/login/ — Obtain access and refresh tokens
  • POST /accounts/refresh/ — Refresh access token
  • POST /accounts/logout/ — Invalidate refresh token

Authorization Header

Authorization: Bearer <access_token>

Rate Limiting

To prevent abuse and ensure fair usage, the API enforces global request limits.

  • Authenticated users: 20 requests per minute
  • Anonymous users: 5 requests per minute

POST /api/images/

Uploads an image and applies processing operations. The image will be processed asynchronously in the background.

Parameters

  • original_image – Image file (jpg, png, webp)
  • operations – JSON describing the transformations

Example Request

{
 "operations": [
  {
   "operation_type": "resize",
   "parameters": {
     "width": 300,
     "height": 300
   }
  },
  {
   "operation_type": "compress",
   "parameters": {
     "quality": 80
   }
  }
 ]
}

GET /api/images/

Returns all images uploaded by the authenticated user.

This endpoint requires authentication.

GET /api/images/{uuid}/

Returns detailed information about a specific uploaded image including:

  • Processing status
  • Image format
  • Creation timestamp
  • Download availability

GET /api/images/{uuid}/download/

Downloads the processed image if processing has completed and the file has not expired.

Image Expiration Policy

Uploaded images have a limited download window to conserve storage resources.

Initial Expiry

  • Authenticated users: 1 hour
  • Anonymous users: 10 minutes

After Download

  • Authenticated users: download remains available for 5 minutes
  • Anonymous users: download remains available for 20 seconds

After this period, the download link will expire and the image may be removed from storage.
